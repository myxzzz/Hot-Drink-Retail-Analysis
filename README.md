## 🎯 核心课题：哪类地段的奶茶店更“抗冻”？

### 1. 业务逻辑 (Business Logic)

* **假设 (Hypothesis)**：在天气变冷、热饮需求（搜索热度）整体上升时，处于“写字楼密集区”的店铺订单比“开放式商圈/风景区”更稳定（更抗冻），因为写字楼是刚需，且有室内取餐环境。
* **分析闭环**：
  * **气象 API** ➡️ 确认寒潮/降温的日期点。
  * **Google Trends API** ➡️ 确认大众对“热饮/奶茶”需求的真实波动。
  * **OSMnx (Overpass API)** ➡️ 获取某城市（如上海、北京或你的家乡）的奶茶店经纬度，并分析其周边的 POI（兴趣点：是公司多，还是公园/旅游景点多）。
